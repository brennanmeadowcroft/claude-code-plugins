# Development Tools

This plugin contains a collection of tools to support development with claude code.

## Targeted Workflow
These tools streamline the use of the following development workflow

1. Work with Claude in planning mode to develop a plan
2. Save the plan to a folder within the `plans` folder at the project root
3. Start other sessions to work on an entire plan or phases of the plan for large projects

## Skills
| Skill                      | Description                                                                                                                             |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| `/build-feature-from-plan` | Reviews the plan and builds the feature according to the plan. Will build specific phases if directed, otherwise will build all phases. |
